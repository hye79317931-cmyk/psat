# PSAT 랜덤 오답노트 PWA

개인용 PSAT 문제앱입니다. 문제 스크린샷과 해설 스크린샷을 붙여넣거나 파일로 올리고, 정답체크·랜덤 풀이·오답 복습·풀이 시간 체크·스타일러스 필기를 할 수 있습니다.

## 포함 기능

- 문제 이미지 등록
- 문제 스크린샷 Ctrl+V 붙여넣기
- 해설 이미지 등록
- 해설 스크린샷 Ctrl+V 붙여넣기
- 정답 ①~⑤ 체크
- 해설 보기
- 전체/오답/오래 걸린 문제/안 푼 문제/다시보기 랜덤 출제
- 문제별 풀이 시간 저장
- 세션 제한시간 설정
- 오답 자동 저장
- 오답 2회 연속 정답 시 자동 해제
- 스타일러스/손가락 필기, 지우개, 확대
- JSON 백업/복원
- PWA 설치 및 오프라인 캐시

## 스크린샷 붙여넣기 사용법

1. 문제등록 탭으로 갑니다.
2. 문제 캡처 후 `문제 스크린샷` 영역을 클릭하고 `Ctrl+V`를 누릅니다.
3. 해설 캡처 후 `해설 스크린샷` 영역을 클릭하고 `Ctrl+V`를 누릅니다.
4. 정답 번호를 고르고 저장합니다.

붙여넣기 버튼도 있지만, 브라우저 권한 제한이 있으면 영역 클릭 후 `Ctrl+V` 방식이 더 안정적입니다. 모바일에서는 파일 선택 방식이 더 안정적입니다.

## 실행 방법

PWA 설치와 오프라인 캐시는 HTTPS 또는 localhost에서 동작합니다.

### GitHub Pages 배포

1. GitHub 저장소에 이 폴더 안의 파일을 그대로 업로드합니다.
2. `Settings > Pages`에서 `Deploy from a branch`, `main`, `/(root)`로 설정합니다.
3. 폰에서 GitHub Pages 주소로 접속합니다.
4. Chrome 메뉴에서 `홈 화면에 추가` 또는 `앱 설치`를 누릅니다.

### PC에서 테스트

이 폴더에서 아래 명령을 실행합니다.

```bash
python -m http.server 8080
```

브라우저에서 `http://localhost:8080`으로 접속합니다.

## 사용 팁

- PSAT 지문이 작으면 문제풀이 화면에서 `＋`를 눌러 확대합니다.
- 필기할 때만 `필기모드 ON`으로 바꾸면 스크롤과 필기가 충돌하지 않습니다.
- 폰 데이터 삭제나 기기 변경 전에는 `통계/백업 > 백업 파일 내보내기`를 꼭 실행합니다.
