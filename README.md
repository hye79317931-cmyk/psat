# PSAT 랜덤 오답노트 PWA v4

개인용 PSAT 문제앱입니다. 문제 스크린샷과 해설 스크린샷을 붙여넣거나 파일로 올리고, 정답체크·랜덤 풀이·오답 복습·풀이 시간 체크·필기앱식 스타일러스 필기를 할 수 있습니다.

## v4 변경사항

- 필기모드 ON/OFF 제거
- 손가락 1개: 문제 화면 이동/스크롤
- 손가락 2개: 핀치 확대/축소
- 스타일러스/마우스: 바로 필기
- 펜 크기와 지우개 크기 각각 조절
- GitHub 자동 동기화 화면 제거
- 스크린샷 붙여넣기 원본 품질 유지

## 포함 기능

- 문제 이미지 등록
- 문제 스크린샷 Ctrl+V 붙여넣기
- 해설 이미지 등록
- 해설 스크린샷 Ctrl+V 붙여넣기
- 정답 ①~⑤ 체크
- 해설 보기
- 전체/오답/오래 걸린 문제/안 푼 문제/다시보기 랜덤 출제
- 문제별 풀이 시간 저장
- 세션 제한시간 설정
- 오답 자동 저장
- 오답 2회 연속 정답 시 자동 해제
- 스타일러스 필기, 지우개, 펜 크기/지우개 크기 조절
- 손가락 이동/핀치 확대
- JSON 백업/복원
- PWA 설치 및 오프라인 캐시

## 필기 사용법

- 손가락으로 화면을 밀면 문제 이미지를 위아래/좌우로 이동합니다.
- 손가락 두 개로 벌리거나 오므리면 확대/축소됩니다.
- 스타일러스 펜이나 PC 마우스로 누르면 바로 필기됩니다.
- `펜`과 `지우개` 버튼으로 도구를 바꿉니다.
- `펜 크기`, `지우개 크기` 슬라이더로 굵기를 조절합니다.

## 스크린샷 붙여넣기 사용법

1. 문제등록 탭으로 갑니다.
2. 문제 캡처 후 `문제 스크린샷` 영역을 클릭하고 `Ctrl+V`를 누릅니다.
3. 해설 캡처 후 `해설 스크린샷` 영역을 클릭하고 `Ctrl+V`를 누릅니다.
4. 정답 번호를 고르고 저장합니다.

붙여넣은 이미지는 JPEG로 변환하거나 축소하지 않고 원본 품질로 저장됩니다.

## GitHub Pages 배포

1. GitHub 저장소에 이 폴더 안의 파일을 그대로 업로드합니다.
2. `Settings > Pages`에서 `Deploy from a branch`, `main`, `/(root)`로 설정합니다.
3. 폰에서 GitHub Pages 주소로 접속합니다.
4. Chrome 메뉴에서 `홈 화면에 추가` 또는 `앱 설치`를 누릅니다.

## PC에서 테스트

```bash
python -m http.server 8080
```

브라우저에서 `http://localhost:8080`으로 접속합니다.


## v6 변경사항
- 문제 풀이 전체화면에서 문제 이미지 영역을 더 크게 확보했습니다.
- 하단 정답 선택/정답 체크 영역을 더 낮고 작게 줄였습니다.
- 현재 문제 시간/세션 시간을 왼쪽 펜·지우개 도구 영역 상단에 배치했습니다.
- 축소 최소값을 5%까지 낮춰 긴 스크린샷도 한 화면에 더 잘 맞출 수 있게 했습니다.
