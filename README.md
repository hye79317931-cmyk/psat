# PSAT 랜덤 오답노트 v11

## v11 수정
- 대분류를 `언어 / 자료 / 상황`으로 정리
- 문제 등록 시 `연도` 입력 추가
- 문제 목록에서 대분류/연도 필터 추가
- 문제풀이 화면에 `언어 랜덤 10/20/30/40문제` 바로 시작 버튼 추가
- 정답 번호를 누르면 즉시 채점 후 자동으로 다음 문제 이동
- 세트 완료 후 결과 화면 표시
- 세트 완료 후 `틀린문제 다시풀기`, `틀린문제 보기`, `전체 다시풀기` 제공
- v10의 3000문제용 이미지 압축/저장 오류 안내 유지

## 업데이트 방법
1. ZIP 압축 풀기
2. GitHub 저장소 `Code → Add file → Upload files`
3. 압축 푼 폴더 안의 파일 전체 업로드
4. `Commit changes`
5. 폰에서 `https://hye79317931-cmyk.github.io/psat/?v=11` 로 접속

## 사용 팁
- 3000문제 이상 넣을 경우 이미지 저장 방식은 `3000문제용 압축 / 용량 작게`를 권장합니다.
- 기존 문제는 그대로 유지됩니다. 기존 `언어논리` 문제도 `언어` 랜덤에 포함되도록 처리했습니다.
- 앱이 예전 화면이면 홈화면 앱 삭제 후 다시 추가하거나 Chrome 사이트 캐시를 삭제하세요.
