## v23 변경사항
- 문제풀기 화면의 언어 랜덤 바로 풀기 영역 제거
- 오답 탭을 복습으로 변경
- 복습을 오답복습 / 오답랜덤복습 / 정답복습으로 분리
- 문제 카드를 누르면 최근 풀이시간, 평균 풀이시간, 누적 풀이시간, 마지막 풀이 시각 표시


## v24 변경사항
- 복습 문제 카드에서 이전 풀이시간 확인 후 바로 복습 풀이 가능
- 복습 풀이 결과에서 이번 풀이시간과 이전 최근/평균 풀이시간 비교 표시
- 오답랜덤복습도 문제별 이전 풀이시간 기준으로 비교 표시
